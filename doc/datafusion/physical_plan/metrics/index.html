  • CompositeMetricsSet
    Collects all metrics during a complex operation, which is composed of multiple steps and each stage reports its statistics separately. Give sort as an example, when the dataset is more significant than available memory, it will report multiple in-mem sort metrics and final merge-sort metrics from SortPreservingMergeStream. Therefore, We need a separation of metrics for which are final metrics (for output_rows accumulation), and which are intermediate metrics that we only account for elapsed_compute time.
